# Chicago Traffic Dashboard 🚦

خريطة تفاعلية لحركة المرور في شيكاغو مع نماذج ML للتنبؤ بالسرعة.

## الملفات المطلوبة على Google Drive
- `processed_traffic_data.csv` — ضع الـ File ID في `Chicago_Dashboard.py`

## التشغيل محلياً
```bash
pip install -r requirements.txt
streamlit run Chicago_Dashboard.py
```
